# WebGL Curl Noise Particles

WebGL2 + Electron による GPU パーティクルアニメーション。約 83 万個（914×914）のパーティクルをテクスチャに位置を焼き込む ping-pong FBO 方式でシミュレーションし、3840×1080（32:9）/ 1920×1080（16:9）の PNG 連番として書き出せる映像制作ツールです。

## 特徴

- **3 つのシミュレーションモード**
  - `Curl Noise` — カールノイズによる流体的な流れ
  - `L-System Roots` — 再帰生成される根状の構造へ収束
  - `PLY Curl` — 任意の点群データへ収束（下記「点群の読み込み」参照）
- **4 種のノイズ場**: Curl / Ridged / Worley / Domain Warped
- **アスペクトモード選択**: 32:9 / 16:9（描画ステージと PNG 書き出しの両方に反映）
- 9 プリセットのカメラ（キー `1`–`9`）＋ドラッグ / ホイールでのオービット操作、キー `0` で回転バースト
- PNG 連番書き出し（フレーム数 / FPS 指定可）

## セットアップ

```bash
npm install
npm start
```

## 点群の読み込み

`PLY Curl` モードは**任意の点群（.ply）**に対応しています。アセットの事前バンドルは不要です。

- HUD の **「Load PLY File」** ボタン、またはキャンバスへの **ドラッグ&ドロップ**で読み込み
- 読み込むと自動で `PLY Curl` モードに切り替わります
- 対応フォーマット: **ASCII / binary（little / big endian）**
- 点数は自動で内部パーティクル数（914×914 = 835,396）に**リサンプル**され、重心を原点に合わせて `BOUNDS` 内に**自動スケール**します
- 色情報（`red`/`green`/`blue`、uchar または float）があれば反映。**色が無い点群は白**で表示します

アプリは点群アセットが無くても起動します（未読込で `PLY Curl` を選ぶとプレースホルダのブロブを表示）。

### バンドルアセット（任意）

`window.__PLY_TARGETS_BASE64__` / `window.__PLY_COLORS_BASE64__`（各 `PARTICLE_COUNT * 3` 要素の Float32 を base64 化）を定義した `ply-targets.js` を置くと、起動時に自動適用されます。本リポジトリにはこのファイルは含まれていません。
